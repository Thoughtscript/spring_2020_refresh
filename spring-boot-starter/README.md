## Setup

Install dependencies, compile, and build the app:

```bash
$ ./gradlew build
```

> This example uses Spring Boot 2.3.3 on Gradle with Groovy.

> See: https://guides.gradle.org/building-java-libraries/

## Spring

```bash
$ ./gradlew run
```
